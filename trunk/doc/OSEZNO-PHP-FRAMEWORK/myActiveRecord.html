myActiveRecord

La clase myActiveRecord es la propuesta que da osezno framework  para acceder a las bases de datos de Postgres, MySql mediante  objetos del Lenguaje de programacion. Extendiendo la clase a subclases  que tengan la informacion de cada una de las tablas a las que se  quieran acceder.
