method waitForCSS [line 574]

void waitForCSS( integer: $insSeconds)

Response command instructing xajax to pause while the CSS files are loaded.

The browser is not typically a multi-threading application, with regards to javascript code. Therefore, the CSS files included or removed with xajaxResponse->includeCSS and xajaxResponse->removeCSS respectively, will not be loaded or removed until the browser regains control from the script. This command returns control back to the browser and pauses the execution of the response until the CSS files, included previously, are loaded.




Tags:

access:  public


Parameters:
